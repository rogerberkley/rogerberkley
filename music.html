Nudge to release a new music track

Let's try this!

What is a Nudge?

A Nudge is a virtual gesture that signals for the release of one of my new music tracks.

How is a Nudge performed?

A Nudge is performed through a digital transaction.

What is the purpose of a Nudge?

The purpose of a Nudge is to signal that a specified upcoming musical release is confirmed for release.

What is a Nudge Window?

A Nudge Window is a timeframe within which a pending musical release is available as accepting Nudges.

What is Nudge Value?

Nudge Value is an amount that is presented with each available Nudge and it is used as minimum value accepted to mark the Nudge as Valid if the Nudge's transaction value matches or exceeds the Nudge Value.

What happens when a Nudge's transaction is confirmed and the Nudge's transaction amount matches the Nudge Value?

If the confirmation occurs during the active Nudge Window and the Nudge's transaction value matches the Nudge Value then the track release will be marked as confirmed and the music track will be released to music platforms.

What happens when a Nudge's transaction is confirmed and the Nudge's transaction amount is less than the Nudge Value?

When a Nudge's transaction amount is less than the Nudge Value the transaction is ignored and has no effect on the available Nudge.

Am I purchasing anything when I make a Nudge?

No, you are not purchasing anything when you make a Nudge.

If I fulfil a Nudge, do I earn any rights to the music track?

No, you do not get any rights to a released music track or any other rights. All rights remain reserved to the original creator.

If a Nudge I made becomes Valid, can I brag about it afterwards?

Yes, you may brag about it afterwards.

What happens if an available Nudge is not fulfilled within the Nudge Window?

The Nudge expires and becomes unavailable, and the track is not released by the end of the Nudge Window. The track may be released any other time after the end of the Nudge Window with or without another Nudge.
